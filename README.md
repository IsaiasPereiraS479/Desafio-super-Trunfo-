#  Super Trunfo - Países

Projeto desenvolvido como parte das atividades da disciplina de Programação em C. O jogo simula uma versão do Super Trunfo com cartas de países, permitindo comparação de atributos e cálculo de um "Super Poder".

## Níveis Implementados

### Nível Aprendiz
- Leitura e exibição de uma carta de país com seus atributos.

### Nível Aventureiro
- Comparação de um atributo escolhido entre duas cartas.

### Nível Mestre

- Comparação completa entre duas cartas.
- Cálculo do **Super Poder**: soma de todos os atributos (inclusive densidade populacional invertida `1/densidade`).
- Comparação atributo a atributo:
  - Para a densidade populacional, vence o **menor valor**.
  - Para os demais atributos (e o Super Poder), vence o **maior valor**.
- Exibição clara dos resultados da comparação.

##  Funcionalidades

- Entrada de dados (nome do país, área, população, PIB etc.).
- Cálculo automático da densidade populacional.
- Cálculo e comparação do Super Poder.
- Comparações de atributos com critérios específicos.
- Saída organizada dos resultados das comparações.
